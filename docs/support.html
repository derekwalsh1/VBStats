Support & Help

Getting Started

Creating Your First Team

  1. Tap the "Add Team" tile on the home screen
  2. Enter your team name
  3. Tap "Create" to save

Starting a Match

  1. Select your team from the home screen
  2. Tap "Add Match" to create a new match
  3. Enter opponent name, match date, and event (optional)
  4. Choose serve/receive (who serves first)
  5. Tap "Create Match"

Recording a Set

  1. From a match, tap "Add Set" to begin
  2. Use the rally outcome buttons to record each point:
    • Kill - Your team wins the rally
    • Error - Your team makes an error
    • Defense - Opponent makes an error or you defend to win
  3. The score updates automatically
  4. Use the undo button to reverse the last rally if needed
  5. Tap timeout buttons (T1/T2) to record timeouts
  6. Set ends automatically when a team reaches winning score with 2-point lead

Managing Data

Editing Teams, Matches, or Sets

Long-press on any item (team, match, or set tile) to delete it. A confirmation dialog will appear to prevent accidental deletions.

Export/Import

  • Export: From a team's match list, use the menu to export all team data as a JSON file
  • Import: Use the menu to import a previously exported JSON file
  • Great for backups or transferring data between devices

Common Questions

Q: What if I tap the wrong rally outcome?

A: Use the undo button (‚Ü∂) in the top-right of the live set screen to reverse the last rally recorded.

Q: Can I track multiple teams?

A: Yes! Create as many teams as you want from the home screen.

Q: Does the app work offline?

A: Yes, completely! All data is stored locally on your device. No internet connection required.

Q: What volleyball format does it use?

A: Standard rally scoring - first to 25 points (win by 2) for sets 1-4, and first to 15 points (win by 2) for set 5.

Q: Can I edit a set after it's completed?

A: Currently, sets cannot be edited after completion. Make sure to use the undo button during the set if you make a mistake.

Troubleshooting

App crashes or freezes

  • Try force-closing and reopening the app
  • Restart your device
  • Ensure you have the latest version installed

Data not showing up

  • Check that you're viewing the correct team
  • Return to the home screen and re-enter the team

Export not working

  • Ensure you have granted file storage permissions
  • Check that you have sufficient storage space
